# AI Engineer Coding Interview Roadmap (8 Weeks – 3 Pass System)

## 🎯 Goal

In 8 weeks:
- Complete Grind 75 + NeetCode 150 THREE TIMES
- Achieve interview-level mastery
- Explain solutions fluently while coding
- Build a strong pattern recognition system

---

## 🔁 Core System: 3 Pass Learning

### Pass 1 — Pattern Learning (Week 1–3)
Focus:
- Understand patterns
- Build intuition
- Learn templates

Rules:
- Can watch solution (after 20–30 min)
- Focus on WHY the solution works

---

### Pass 2 — Reinforcement (Week 4–6)
Focus:
- Solve without help
- Strengthen memory
- Improve speed

Rules:
- NO solution watching
- Reconstruct from memory

---

### Pass 3 — Interview Mode (Week 7–8)
Focus:
- Communication
- Speed
- Real interview simulation

Rules:
- Talk while coding
- Time pressure
- Clean code

---

## 🧠 Pattern-Based Learning (NeetCode Map)

1. Arrays & Hashing
2. Two Pointers
3. Sliding Window
4. Stack
5. Binary Search
6. Linked List
7. Trees
8. Heap / Priority Queue
9. Backtracking
10. Graph
11. DP (1D + 2D)
12. Greedy / Intervals

---

## 📚 Problem Sources

- Blind 75 (Core)
- NeetCode 150 (Full coverage)

---

## 🔁 Repetition Schedule

Each problem must be solved:
- Day 1 → Learn
- Day 2 → Re-solve
- Day 5 → Re-solve
- Day 10 → Re-solve

---

## 🧪 Daily System

1. New problems (pattern-based)
2. Repetition (old problems)
3. Verbal explanation
4. Mistake logging

---

## 🚨 Success Criteria

You are ready if:

- Medium ≤ 20 min
- Pattern identified ≤ 2 min
- Can explain clearly while coding
- No major blind spots

---

## 🔥 Core Philosophy

"Train like it's an interview, not a homework session."
